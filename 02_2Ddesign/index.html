<div class="textcontainer"> <p class="margin"> </p> <h3>Week 2: 2D Design & Cutting</h3> <p class="margin"> </p> <div class="flexrow"> <a id="btn" href="./CAD.zip" download> Download Button </a> </div> <p class="margin"> </p> <h4>Assignment 1: Make a Box</h4> <p>For this assignment, I followed a tutorial from Hobbyist Maker on automatically sized finger joint boxes: https://www.youtube.com/watch?v=9U2JPfkQpsE. I customized the number of fingers on each side to reduce kerf and chose the dimensions 350mm x 250mm x 150mm because those were the average dimensions of a male shoebox. I used 6mm wide cardboard for the material and laser-cutting portion. </p> <p></p> <image style = "width: 75%"src = "./Fingerjoint Box.png" alt = "week 2 image"> </image> <p></p> <image style = "width: 75%"src = "./pieces.JPG" alt = "week 2 image" class="rotate270"> </image> <image style = "width: 75%"src = "./assembled.JPG" alt = "week 2 image"> </image> After putting the box together, I found that the fit was still a bit loose likely due to kerf, so I added some hot glue to the joints to make it more secure. Overall, this was my first time using Fusion 360/CAD software (apart from the lab tutorial) as well as a laser cutter, and I felt so awesome trying both technologies out! Pretty satisfied as well with the ultimate result given that it is my first time :D <image style = "width: 75%"src = "./glue.JPG" alt = "week 2 image"> </image> </p> <h4>Assignment 2: Fusion 360 Modeling and Assembly</h4> <p> The two models additional Fusion360 I made were a AA battery and a nut/bolt. Using the external jaws of the calipar, I found that the diameter was approximately 14mm. I followed this tutorial from Youtuber CAD Jungle: https://www.youtube.com/watch?v=9ga4mE6EACo to make the battery. I tried to make the appearance more battery-like by switching up the colors of the surfaces. <image style = "width: 75%"src = "./AA Battery.png" alt = "AA Battery"> </image> Next, to make the nut and bolt, I measured the diameter of the bolt to be approximately 3mm. I followed a Youtube tutorial from The Fusion Essentials: https://www.youtube.com/watch?v=dt5W-4y8r5I. I found this tutorial to be super helpful because it explained how to use the thread tool in Fusion 360, which I had no idea about before. I also learned how to use the circular pattern tool to make the hexagonal shape of the nut. <image style = "width: 75%"src = "./Nut and Bolt.png" alt = "Nut and Bolt"> </image> I then created a new project that assembled these pieces: the box, the battery, and the nut and bolt. I scaled the battery and nut/bolt up to be visible then moved them fit in the box. I then deselected some of the sides of the box so that the two objects were still visible inside the assembly. <image style = "width: 75%"src = "./assembly.png" alt = "Assembly"> </image> </p> </div>
